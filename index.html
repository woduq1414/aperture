Aperture란?

여러분이 채워나갈 역사

올해는 무엇을 할 거냐면요..

2020년의 Aperture는 기본적으로 웹 프론트엔드 개발부/백엔드 개발부, 웹 크롤링부로 분과를 나눠 진행될 예정입니다.

프론트엔드

프론트엔드부에서는 css, Javascript를 배우면서 프론트엔드 개발을 익히고 더 나아가 세계에서 가장 많이 사용하는 프론트엔드 라이브러리인 React의 기초까지 배웁니다.

백엔드

백엔드부에서는 Python의 백엔드 개발 라이브러리인 Flask와 Django를 학습하며 서버 구축, Database 등을 배웁니다.

웹 크롤링

웹 크롤링부에서는 Python을 이용해 인터넷 상의 다양한 정보들을 자동으로 수집해 데이터의 활용 방안을 탐구합니다.

세 개 분과의 협력으로 동아리 프로젝트도 기획하고 있습니다.
